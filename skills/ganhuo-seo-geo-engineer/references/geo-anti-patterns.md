# GEO Anti-Patterns

## Do Not Fabricate Proof

Do not invent:

- statistics;
- sample sizes;
- customers;
- expert quotes;
- product performance;
- rankings;
- search volume;
- conversion lift;
- AI citation frequency.

If the user asks for fake proof, refuse that part and create a backlog item for real proof.

## Do Not Promise Rankings

Acceptable:

```text
This rebuild improves structure, answerability, and citation readiness.
```

Not acceptable:

```text
This will make ChatGPT cite the page.
```

## Do Not Stuff Keywords

Repeating the same phrase does not make content GEO-friendly. Prefer entities, questions, concrete steps, limitations, and reusable answer blocks.

## Do Not Erase the Original Voice

Preserve the original point of view unless the user explicitly asks for a new positioning. A GEO rebuild is not a generic brand rewrite.

## Do Not Overbuild Thin Material

If the supplied material is too thin, return a diagnosis and a supplement backlog before producing a polished article.
