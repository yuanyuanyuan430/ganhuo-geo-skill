# Prompts for ganhuo-geo-skill

这些 prompt 可以直接复制到 Codex / 干活 AI 中使用，用来把旧文章、产品页、教程、FAQ 和知识库内容改造成 GEO / AI 搜索友好的内容资产。

## 1. 旧文 GEO 改造

```text
使用 ganhuo-geo-engineer 改造下面这篇旧文。

目标：
把它改成 GEO / AI 搜索友好的内容资产。

目标用户：
希望被 AI 推荐的场景：
原文：

请输出：
1. Core Answer：AI 可以直接摘取的一句话答案；
2. Rewritten Content：改造后正文；
3. Entity Map：核心实体、概念、产品、用户和场景；
4. FAQ：真实用户会问的问题；
5. GEO Scorecard：答案前置、实体清晰、证据承载、FAQ 完整、段落可引用等评分；
6. Supplement Backlog：最值得补的数据、案例、截图、用户问题和测试记录；
7. Publishing Notes：标题、摘要、内链、更新时间和分发建议。
```

## 2. 产品页 AI 推荐改造

```text
使用 ganhuo-geo-engineer，帮我把下面这个产品页改造成更适合 AI 搜索推荐的页面。

产品：
目标用户：
主要竞品：
希望被推荐的场景：
当前页面内容：

请重点加强：
1. 产品是什么、适合谁、解决什么问题；
2. 与竞品相比的可验证差异；
3. AI 容易摘取的摘要、表格、FAQ；
4. 需要补充的证据和用户案例；
5. 页面标题、H2、schema、内链建议。
```

## 3. 教程 / 文档改造
